# Security Policy

## Supported Versions

Only the latest version of ProofOfBuilder is supported for security updates.

## Reporting a Vulnerability

We take the security of our project seriously. If you find a potential vulnerability, please report it privately to the maintainers rather than opening a public issue.

You can contact us via:
- Discord: [Stacks Discord](https://discord.gg/stacks)
- GitHub: [Private Reporting](https://github.com/Earnwithalee7890/Stacks-Reputation-Rewards/security/advisories/new)

We will respond within 48-72 hours to acknowledge your report and discuss the next steps.
